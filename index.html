Sintetizador do NEWAVE

Versão: 2.2.0

Data: 22/01/2025

O sintetizador-newave é uma aplicação CLI para formatação dos arquivos de saída do modelo NEWAVE. O NEWAVE é desenvolvido pelo CEPEL e utilizado para a planejamento da operação do Sistema Interligado Nacional (SIN).
